Screen sizes and bootstrap infixes mappings
Screen Size Bootstrap Class Infix Screen Dimention
Extra small None 0px to 576px (it means jo devices 0 se 576px ke beech mein honge unko Extra small devices bolenge)
Small sm 576px to 768px
Medium md 768px to 992px
Large lg 992px to 1200px
Extra large xl 1200px to 1400px
Extra extra large xxl 1400px and above

Inner Columns example
Remember that Inner Column cannot directly go inside Outside Column
Container > Row > Outer Column > Inner Row > Inner Column > Content (Text or Images)
yahan pe Col1 ke andar 4 inner columns hai inner-col1,2,3,4
Outer Col1,Col2,Col3,Col4 ki width 6,2,2,2 set ki gayi hai for large devices (my laptop device)
inner col1,col2,col3,col4 ki width 3,3,3,3 set ki gayi hai for large devices
ab jab mein screen size ko medium tak reduce karunga tab sabhi Outer columns ki width 12 set ho jayegi by default
aur inner columns col1,2,3,4 ki 12,4,4,4 set ho jayegi (jaisa ki maine set kiya hai)
